# C Final Project

## Overview
This project is a menu-driven C application that allows users to perform calculations, read from files, and write results to files. The program is modular and organized into multiple header files for maintainability.

## Features
- Interactive menu system  
- File input and output  
- Multiple calculation modules  
- Structured main loop for user control  

## Files Included
- main.c – Main application logic  
- menu.h – Menu display and input handling  
- mainLoop.h – Controls program flow  
- calc1.h / calc2.h – Calculation modules  
- fileRead.h – File reading functions  
- fileWrite.h – File writing functions  

## How to Compile
```bash
gcc main.c -o finalProject
./finalProject
